# Debt Tracker Pro 📱

تطبيق احترافي لإدارة الديون — Android 13+

## المميزات
- 🔐 حماية بـ PIN مشفر
- 💰 إدارة الديون (عليك / لك) بالدرهم المغربي
- 📊 إحصائيات مع Pie Chart
- 📥 تصدير Excel (xlsx)
- 🔍 بحث وفلترة وترتيب
- 🎨 تصميم احترافي Material Design 3

## بناء APK
1. ارفع على GitHub
2. Actions → Build APK → Run workflow
3. حمّل APK من Artifacts
